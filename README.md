# HTML Cloak

A simple HTML-based URL launcher project hosted with GitHub Pages.

## Disclaimer

This project was not originally created by me.  
All credit goes to the original author of the project.

The only modification made in this version is the removal of the requirement to manually enter `https://` before URLs.

No ownership of the original design, code, or functionality is claimed.

## Website

Visit the hosted version:

https://yawnyawnpizza.github.io/html-cloak/

## Features

- Opens URLs through an about:blank-style page
- URL input interface
- Automatically handles missing URL protocols
- Simple dark-themed UI
- GitHub Pages compatible

## Usage

1. Open the website
2. Enter a URL
3. Click **Generate**
4. The page will open in a new window

## Hosting

This project is hosted using GitHub Pages.
